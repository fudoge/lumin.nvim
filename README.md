# lumin-nvim

A Neovim theme, ported from [frypan05/Lumin](https://github.com/frypan-5/Lumin)

## Table

cursor                 → Cursor

syntax.comment         → Comment, @comment
syntax.string          → String, @string
syntax.keyword         → Keyword, @keyword
syntax.function        → Function, @function
syntax.type            → Type, @type
syntax.variable        → Identifier, @variable
